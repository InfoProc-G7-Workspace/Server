# Server

## Test Environment Setup

Run each of these in a separate terminal:

1. **Web server**
   ```bash
   cd backend && npm run dev
   ```

2. **Robot simulator**
   ```bash
   node robot_sim/robot-simulator.js
   ```

3. **Robot camera**
   ```bash
   python3 robot_sim/robot_camera.py
   ```

4. **Gaussian processor**
   ```bash
   node gaussian_proc_sim/processor.js
   ```
